1.- Que es box model

Se refiere al area donde se muestra el contenido cuyo tamanio es modificable

2.- Que es una caja en box model

Es lo que representa al elemento html

3.- Elementos y/o colores y/o capas de una caja box model (definicion y ejemplo)

Los elementos son 3 y represetandos por los siguientes colores:

Margin Border Padding

4.- Que son las propiedades width y height en un box model

La propiedad Width representa el ancho de los elementos

La propiedad Height representa la altura de los elementos

5.- Cual es la importancia de usar box-sizing: border-box y con que selector se utiliza

La importancia de utilizar box-sizing: border-box es para que todos los elementos html puedan ser modificados libremente

La forma de utilizar es mediante el selector universal * {box-sizing: border-box }

6.- Definicion y ejemplo de PADDING

Padding: Propiedad que establece el area de relleno en los cuadrados de los elementos

Content

7.- Definicion y ejemplo de BORDER

Border: Propiedad que aniade tamanio a la parte final del contenido

Content

8.- Definicion y ejemplo de MARGIN

Margin: Propiedad que aniade el tamanio por la parte de afuera de la caja

9.- Que es colapsado de margenes verticales(Concepto y ejemplo)

El colapsado de margenes sucede cuando 2 elementos tienen un tamanio mayor o menor

Al momento de ser interceptados por los margenes se utiliza el que tiene mayor tamanio

10.- Que es el margin negativo

El margin negativo es cuando los elementos se mueven diferente a lo usual
